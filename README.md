# gomezaldaz.com

Sitio web oficial del autor **Olmo Gómez Aldaz**.

Web estática alojada en **GitHub Pages**, con dominio personalizado.
El sitio articula tres ejes al mismo nivel:
- Sentencia (hecho jurídico)
- Libros (obra publicada)
- Autor

## Estructura del sitio

- `/` → Home
- `/sentencia.html` → Hecho jurídico central y referencias públicas
- `/libros.html` → Obras publicadas (castellano / inglés)
- `/autor.html` → Información del autor

Estructura preparada para versión futura en inglés (`/en/`).
